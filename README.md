# 📐 Linear Algebra for AI/ML

> Learning Linear Algebra by Building Visualizations in Python

This repository documents my journey of learning **Linear Algebra** by implementing mathematical concepts from scratch and visualizing them using Python.

Instead of only studying theory, I build interactive visualizations that help develop an intuitive understanding of the mathematics behind Artificial Intelligence, Machine Learning, and Data Science.

---

## 🎯 Goal

The purpose of this repository is to:

- Learn Linear Algebra from first principles.
- Visualize mathematical concepts instead of memorizing them.
- Strengthen Python programming skills.
- Build a strong mathematical foundation for AI/ML.
- Create an open portfolio of my learning journey.

# ✨ Current Project

## 2D & 3D Vector Visualizer

### Features

- Accepts user-defined vectors
- Plots vectors from the origin
- Supports both 2D and 3D vectors
- Automatically adjusts graph limits
- Checks linear independence using the determinant (2D)
- Displays vector endpoints
- Saves the generated visualization
- Uses equal axis scaling
